# Tool Installation

## 1. Homebrew

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## 2. GitHub CLI

```bash
brew install gh
```

## 3. Node and pnpm

```bash
brew install node
brew install pnpm
```

## 4. Pocketbase

```bash
brew install pocketbase
```

## 5. Vibe Launcher

```bash
brew install pykancha/tap/vibe-launcher
```
